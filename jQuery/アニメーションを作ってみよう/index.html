    2. JavaScript (jQuery) ではwebページにアニメーションをつけることができます。

  2. fadeOutメソッドを実装しよう

    3. fadeOutメソッドは割り当てたセレクタをフェードアウトさせるアニメーションです。
    fadeOutメソッドはjsファイルを用いて以下のように記述します。

    引用 : https://prog-8.com/jquery


    fadeOutメソッドは【$('セレクタ').fadeOut();】と記述します。
    試しに以下のコードを次の画像に割り当てて実装してみます。 

    
                $(document).ready(function () {
                  $('Luke_Sleep').fadeOut();
                });
              
    
                id="Luke_Sleep"

    しかしこの状態ではページが開いた瞬間にフェードアウトしてワンちゃんは消えてしまいますね(F5で確認)
    それを解決するために次の説明に進みましょう！
