  3. HTML構造（基本形）

    4. 以下に今回作成するページのHTML例を示します。 

    <!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>My Simple Website</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>

  <header>
    <div class="container">
      <h1 class="logo">My Site</h1>
      <nav class="nav">
        <ul>
          <li><a href="#">ホーム</a></li>
          <li><a href="#">サービス</a></li>
          <li><a href="#">お問い合わせ</a></li>
        </ul>
      </nav>
    </div>
  </header>

  <section class="hero">
    <h2>ようこそ！</h2>
    <p>これはサンプルのWebページです。</p>
  </section>

  <section class="features">
    <div class="feature">特徴1</div>
    <div class="feature">特徴2</div>
    <div class="feature">特徴3</div>
  </section>

  <footer>
    <p>© 2025 My Site. All rights reserved.</p>
  </footer>

</body>
</html>

  4. CSS(style.css)

    5. 以下に今回作成するページのCSS例を示します。 

    body {
  margin: 0;
  font-family: 'Helvetica', sans-serif;
  background: #f9f9f9;
}

header {
  background: #2a7ae2;
  color: white;
  padding: 20px 0;
}

.container {
  width: 90%;
  max-width: 1000px;
  margin: 0 auto;
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.logo {
  font-size: 24px;
}

.nav ul {
  display: flex;
  list-style: none;
  padding: 0;
}

.nav li {
  margin-left: 20px;
}

.nav a {
  color: white;
  text-decoration: none;
}

.hero {
  background: #e2edff;
  text-align: center;
  padding: 60px 20px;
}

.hero h2 {
  font-size: 32px;
  margin-bottom: 10px;
}

.features {
  display: flex;
  justify-content: space-between;
  gap: 20px;
  padding: 40px 20px;
  max-width: 1000px;
  margin: 0 auto;
}

.feature {
  flex: 1;
  background: white;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
  border-radius: 8px;
}

footer {
  background: #333;
  color: white;
  text-align: center;
  padding: 20px;
}

  5. 要素の補足

    6. 以下にサンプルファイルに示した要素などの詳細について示します↓

    要素：内容

    要素 内容
    .container 幅を制限した中央揃え用ラッパー（Flexbox使用）
    .hero メインビジュアル的なセクション
    .features Flexboxを使って3つの特徴を横並びに
    footer サイトの最下部に固定された簡易的なフッター
