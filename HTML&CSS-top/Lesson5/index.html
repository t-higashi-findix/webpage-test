Lesson 5：ページ構成の設計

ヘッダー・フッター・ナビ… Webページの“型”を理解しよう！

背景画像 著作者：rawpixel.com/出典：Freepik

    目次

  1. はじめに
  2. Webページの一般的な構成
  3. 各パーツの役割
  4. サンプル：基本的なページ構成
  5. 練習課題
  6. まとめ

  1. はじめに

    2. これまでのレッスンで、HTMLタグやCSSを使って見た目を整える方法を学んできました。
    今回は、それらをどのように組み合わせて1つのページを構成するか、つまりWebページの設計について学んでいきます。
    実際のサイトを見ると、どれも一定の「型」に沿って作られていることがわかるはずです。


  2. Webページの一般的な構成

    3. 多くのWebページは、以下のようなブロックで構成されます


  3. 各パーツの役割

    4. 各タグの役割について説明します。

    タグ名 役割
    <header> サイトのタイトル、ロゴ、メニューなど
    <nav> ページ内や外部へのナビゲーション（リンク）
    <main> ページの主なコンテンツ部分
    <footer> コピーライト、連絡先などの末尾情報

  4. サンプル：基本的なページ構成

    5. ページの構成について以下にサンプルコードを示します。

    <!DOCTYPE html>
<html>
<head>
 <meta charset="utf-8">
 <title>サンプルページ構成</title>
 <link rel="stylesheet" href="style.css">
</head>
<body>
 <header>
  <h1>My Web Site</h1>
 </header>

 <nav>
  <ul>
   <li><a href="#">ホーム</a></li>
   <li><a href="#">プロフィール</a></li>
   <li><a href="#">お問い合わせ</a></li>
  </ul>
 </nav>
 <main>
  <h2>はじめまして</h2>
  <p>このページはHTMLとCSSの練習用です。</p>
 </main>
 <footer>
  <p>© 2025 My Web Site</p>
 </footer>
</body>
</html>
    body {
 font-family: sans-serif;
 margin: 0;
 padding: 0;
}

header, nav, main, footer {
 padding: 20px;
 text-align: center;
}

header {
 background-color: #2a7ae2;
 color: white;
}

nav {
 background-color: #e0e0e0;
}

nav ul {
 list-style: none;
 padding: 0;
 display: flex;
 justify-content: center;
}

nav li {
 margin: 0 15px;
}

nav a {
 text-decoration: none;
 color: #333;
}

main {
 background-color: #f9f9f9;
 min-height: 300px;
}

footer {
 background-color: #ddd;
 font-size: 0.8em;
}

    ポイント解説

    • header, nav, main, footerはHTML5のセクショナル要素
    • ul と li を使ってナビゲーションのリストを作成
    • レイアウトは基本的に縦並び（後ほどFlexboxで横並びにも挑戦！）

  5. 練習課題

    6. 以下の条件で、あなただけの簡易Webページ構成を完成させてみましょう

    • ページに header, nav, main, footer を使う
    • ナビゲーションに3つ以上のリンクを追加する
    • 全体をCSSで見やすく整える

  6. まとめ

    7. 今回のまとめです↓

    • Webページには一定の構成パターン（型）がある
    • header, nav, main, footer を使い分けよう
    • セクションごとに CSS で背景や文字色を変えると見やすくなる
    • ul+li+aでナビゲーションリンクを作成する

    次のページではボックスを横に並べたり中央揃えにしたりといった、自由なレイアウトが可能になる「Flexbox（フレックスボックス）」について学びます！
