Lesson 1：HTMLの超基本

HTMLってなに？という方も安心。

まずはページの土台づくりから始めましょう！

  1. はじめに

    2. インターネットで見られるWebページって、そもそもどうやってできているのでしょう？
    実はその土台となっているのが「HTML（エイチティーエムエル）」という言語です。
    このレッスンでは、HTMLの基本構造と、最初に覚えるべき見出しや段落の使い方を学びます。
    これができるようになると、あなたも「自分でページが書けた！」という第一歩を踏み出せます。


  2. HTMLってなに？

    3. HTMLは、HyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）の略で、
    Webページの**構造（骨組み）**を記述するための言語です。
    ページの見た目を整えるCSSとは違い、HTMLは「この部分は見出し」「ここは段落」「ここに画像を表示する」といった内容の役割を記述します。


  3. HTMLの基本構造

    4. まずは最もシンプルなHTML文書の基本構造を見てみましょう。
    左が実際のhtmlのプログラムで、右が実行結果です

    
    <!DOCTYPE html>
    <html>
    <head>
        <title>はじめてのページ</title>
    </head>
    <body>
        <h1>こんにちは！</h1>
        <p>これが最初の文章です。</p>
    </body>
    </html>

    これだけではわかりませんね。
    各行の内容について簡単に解説を加えます。

    <!DOCTYPE html>
    		 ：HTML5を使うよ、という宣言（おまじない） 
    <html>～</html>
    		 ：HTML文書全体を囲むタグ 
    <head>～</head>
    		 ：ページの設定やタイトルを記述 
    <title>～</title>
    		 ：タブに表示されるページのタイトル 
    <body>～</body>
    		 ：実際に画面に表示される部分

    htmlでは＜～＞と＜／～＞の"タグ"という部分に囲まれたもので作られています。
    タグを使うことでページを領域ごとに分割しているイメージです


  4. 見出しと段落を書いてみよう

    5. 見出しタグ

    HTMLでは、見出しを以下のように表現します。数字が小さいほど大きな見出しです。
    左が実際のhtmlのプログラムで、右が実行結果です

    
    <h1>大見出し</h1>
    <h2>中見出し</h2>
    <h3>小見出し</h3>

    HTMLでは上記のような見出しタグを使ってHPの見出しを作ります。

    次は段落タグです。

    段落タグ

    HTMLでは段落を以下のように表現します。
    本文となる文章によく使われます。

    
    <p>これは段落です。何かを説明したりする文章に使われます。</p>

  5. 実際に書いてみよう！

    6. 以下のコードをコピーして、自分のエディタなどで試してみてください。

    
    <!DOCTYPE html>
    <html>
    <head>
        <title>自己紹介</title>
    </head>
    <body>
        <h1>こんにちは！</h1>
        <p>はじめまして。私は Web開発を勉強中です。</p>
        <h2>こんにちは！</h2>
        <p>映画を見ることと、散歩することが好きです。</p>
    </body>
    </html>

  6. 練習課題

    7. 以下の項目を取り入れて、あなただけの自己紹介ページを作ってみましょう

    • 自分の名前
    • 好きな食べ物
    • 最近見た映画
    • 見出しを2つ以上、段落を3つ以上使う

  7. まとめ

    8. 今回のまとめです↓

    • HTMLはWebページの構造を作るための言語
    • html, head, body の3つが基本構成
    • h1〜h6で見出し、pで段落を書く
    • 実際にコードを書いてみることが一番の近道！

    次のページではWebページにリンクや画像を追加する方法を紹介します！
    「他のページに飛ぶ」「画像を表示する」といった基本操作をマスターしましょう。
