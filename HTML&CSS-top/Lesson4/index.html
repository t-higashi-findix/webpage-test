Lesson 4：ボックスとレイアウトの基礎

Webページの“並び方”を理解して、自由に配置してみよう

背景画像 著作者：rawpixel.com/出典：Freepik

    目次

  1. はじめに
  2. 全ての要素は「ボックス」
  3. レイアウトを整えるCSSプロパティ
  4. 実例：ボックスの見た目を調整
  5. 要素を横に並べるには？
  6. 練習課題
  7. まとめ

  1. はじめに

    2. 前回のレッスンでは、文字に色やサイズなどのスタイルを加えてページにデザインをつける方法を学びました。
    今回のテーマは、**要素の配置や並び方を整える「レイアウト」**です。
    CSSでは、ページ内の各要素を「ボックス」として捉えて、
    そのサイズ・位置・余白などを調整することで、見やすいページ構成を作っていきます。


  2. すべての要素は「ボックス」

    3. HTMLの要素は、**見えない“箱（ボックス）”**に包まれています。
    これを「ボックスモデル」と呼びます。

    引用：Progate(https://prog-8.com/)


  3. レイアウトを整えるCSSプロパティ

    4. よく使うCSSのプロパティをいくつかご紹介します↓

    プロパティ 内容 例
    width 横幅 width: 300px;
    height 高さ height: 150px;
    padding 内側の余白 padding: 20px;
    margin 外側の余白 margin: 10px;
    border 枠線 border: 1px solid #ccc;
    display 表示形式（block, inlineなど） display: block;
    box-sizing 枠線やパディングの計算方法 box-sizing: border-box

  4. 実例：ボックスの見た目を調整

    5. ボックスの見た目を調整する実例についてお見せします。

    
<!DOCTYPE html>
<html>
<head>
  <title>box model</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="box">ここはボックスです</div>
</body>
</html>

    
.box {
  width: 300px;
  padding: 20px;
  margin: 20px auto;
  border: 2px solid #2a7ae2;
  background-color: #f0f8ff;
  text-align: center;
  box-sizing: border-box;
}

  5. 要素を横に並べるには？

    6. よく使うCSSのプロパティをいくつかご紹介します↓

    プロパティ 説明 例
    color 文字の色を指定 coler: blue;
    font-size 文字の大きさを指定 font-size: 24px;
    font-family フォントの種類を指定 font-family: sans-serif;
    font-weight 太字などの強調（boldなど） font-weight: bold;
    text-align 文字の配置（left,centerなど） text-align: center;

  6. 練習課題

    7. 以下の項目に取り組み、Lesson4の復習をしてみましょう

    • div要素に枠線、背景色、内側と外側の余白をつける
    • 同じスタイルのボックスを2つ並べて、カード風にする
    • box-sizing: border-boxを使ってサイズの扱いを確認する

  7. まとめ

    8. 今回のまとめです↓

    • HTML要素は「見えない箱（ボックス）」で構成されている
    • ボックスモデルの構造（margin, padding, border, content）を理解しよう
    • width, height, padding, margin, borderで大きさや位置を調整できる
    • 要素を横並びにするにはfloatやflexboxを使う

    次のページでは、ヘッダー・フッター・ナビケーションなど、実践的なWebページの構成方法について学んでいきます
