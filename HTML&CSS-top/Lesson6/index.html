Lesson 6：Flexboxで要素を並べる

横並び・中央揃え・レスポンシブ…レイアウトの悩みはFlexboxで解決！

背景画像 著作者：rawpixel.com/出典：Freepik

    目次

  1. はじめに
  2. Flexboxとは？
  3. 基本の使い方
  4. よく使うFlexboxプロパティ
  5. 実例：カードを横並びに
  6. レスポンシブ対応のヒント

  8. まとめ

  1. はじめに

    2. 前回は、Webページ全体の構成（ヘッダー、ナビ、フッターなど）を学びました。
    今回はその中でも特に重要な「レイアウト」の技術として、Flexbox（フレックスボックス）を学びます。
    Flexboxを使うことで、要素の横並びや中央揃え、間隔の調整などがとても簡単になります。


  2. Flexboxとは？

    3. Flexbox（Flexible Box Layout）は、要素を効率よく並べるためのCSSのレイアウト手法です。

    • 要素を横に並べたい
    • 要素を上下左右中央に配置したい
    • 要素間にスペースを入れたい
    こんな場面で大活躍です。

    ↑私はこのように要素を横並びにするために使うことが多いです！


  3. 基本の使い方

    4. 基本的な使い方は以下の通りです。
    CSSファイルに横並びにしたい要素を含んだ親要素にdisplay: flex;と記述します。 

    セレクタ {
  display: flex;
}

    これでセレクタに含まれる子要素が横並びになります。 例として、以下に3つの要素を横に並べるコードを示します。 

    <!DOCTYPE html>
<html>
<head>
  <title>box model</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="container">
    <div class="box">A</div>
    <div class="box">B</div>
    <div class="box">C</div>
  </div>
</body>
</html>
    .container {
  display: flex;
}

.box {
  width: 100px;
  height: 100px;
  background-color: #2a7ae2;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
  border: 3px solid black;
}

  4. よく使うFlexboxプロパティ

    5. よく使うFlexboxプロパティを以下にまとめます。

    プロパティ

    プロパティ 説明 主に使う場所
    display: flex; Flexboxを有効にする 親要素（コンテナ）
    flex-direction 並び方（row=横、column=縦） 親要素
    jusify-content 主軸方向の並び方 親要素
    align-items 交差軸方向の並び方 親要素
    gap 要素間のスペース 親要素
    align-self 要素ごとに縦位置を変える 子要素

    配置のパターン

    値 説明
    flex-start 左揃え（初期状態）
    center 中央揃え
    space-between 両端揃え＋間に均等なスペース
    space-around 全体に均等なスペース
    space-evenly 等間隔で配置

  5. 実例：カードを横並びに

    6. 実例として3枚のカードを横並びにするコードを以下に示します。 

    <!DOCTYPE html>
<html>
<head>
  <title>box model</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <div class="card-container">
    <div class="card">カード1</div>
    <div class="card">カード2</div>
    <div class="card">カード3</div>
  </div>
</body>
</html>
    .card-container {
  display: flex;
  gap: 20px;
  justify-content: center;
}

.card {
  width: 150px;
  height: 100px;
  background-color: #f2f2f2;
  border: 1px solid #ccc;
  text-align: center;
  line-height: 100px;
  font-weight: bold;
}

    Flexboxを使うと要素を簡単に横並びにすることができますね。
    ちなみにgap: 20px;で要素間のスペースを調節しています。marginなどで調整が難しいときなど便利です。

    レスポンシブ対応のヒント

    Flexboxは画面サイズにも柔軟に対応します。
    次のようにcssファイルに書き加えると、要素が折り返されるようにできます。

    .card-container {
  display: flex;
  gap: 20px;
  justify-content: center;
}

.card {
  width: 150px;
  height: 100px;
  background-color: #f2f2f2;
  border: 1px solid #ccc;
  text-align: center;
  line-height: 100px;
  font-weight: bold;
}

.card-container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

    flex-wrap: wrap;で要素を折り返すことができるようになります。
    （拡大などをすることで3つの要素すべてが画面横いっぱいに収まらなくなり、折り返されることが確認できます）


  6. 練習課題

    7. 以下の項目を取り入れて、Lesson6の復習をしましょう

    • 要素を3つ横並びにして、間隔をつけて中央揃え
    • 要素を縦方向に並べて、上下中央に配置
    • 画面が狭いときに折り返すレイアウト


  7. まとめ

    8. 今回のまとめです↓

    • display: flex;で横並びレイアウトが簡単にできる
    • justify-contentやalign-itemsで位置を調整できる
    • gapで要素間のスペースも自由自在
    • flex-wrapを使えばレスポンシブにも対応できる
