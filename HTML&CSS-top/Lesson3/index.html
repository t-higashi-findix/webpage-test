Lesson 3：CSSの基本と文字装飾

  1. はじめに

    2. HTMLでWebページの「構造」が書けるようになったら、次に学ぶべきはCSS（シーエスエス）。
    CSSは、文字の色や大きさ、背景などを設定できる「Webページのデザイン担当」です。
    このレッスンでは、CSSの基本的な使い方と、文字にスタイルを加える方法を学びます。
    ほんの少しCSSを加えるだけで、ページの印象が大きく変わりますよ


  2. CSSってなに？

    3. CSSは Cascading Style Sheets（カスケーディング・スタイル・シート）の略で、HTMLで作った構造に対して「見た目の指示」を与える言語です。
    例えば、文字の色やサイズ、フォント、行間、背景などを変えることができます。


  3. CSSの書き方（基本構文）

    4. CSSは○○.cssというファイル形式です。
    また、実際のCSSファイルの書き方は以下の構文を基本として作られています。


    セレクタ {
  プロパティ: 値;
}
    h1 {
  color: red;
  font-size: 32px;
}

    左が基本構文です。セレクタとはどのHTMLの要素にCSSを適用するか決める部分です。
    右側はCSSの書き方の一例です。 この例では読込先の.htmlファイルのh1セレクタに対して、
    colorをredに（文字の色を赤に）、font-sizeを32px（フォントサイズを32ピクセル）に設定するものとなっています。
    h1、つまり見出しを装飾しているという事です。


  4. CSSの適用方法

    5. CSSは3つの方法でHTMLに適用できます。

    • HTMLファイルとは別にCSSファイルをつくり、
      HTML文書内にそのファイルをCSSとして利用することを記す。
      （外部スタイルシート）
    • HTML文書のhead要素内にstyle要素を設け、HTML文書内にCSSを埋め込む。
      （内部スタイルシート）
    • HTMLの各要素にstyle属性を用いて直接記入する。
    引用：「HTML+CSSまとめページ/CSSの適用方法」(https://cc.musabi.ac.jp/kenkyu/cl1/html5/css_apply.html)


    この記事は外部スタイルシートを使って.cssファイルを読み込むことで.htmlファイルを装飾しています。
    以下にそれぞれの方法を用いたCSSの適用方法について示します。

    • 外部スタイルシートの方法

      • .cssファイルを.htmlファイルから読み込んで読込先の.htmlファイルの装飾をします。
      他のhtmlファイルに使いまわせるといったメリットがあります
      .htmlファイルで.cssファイルを読み込むにはheaderタグで次のように宣言する必要があります。
      <link rel="stylesheet" href="○○.css">
      以下は.cssファイルを読み込まない状態における画面です。 

      <!DOCTYPE html>
<html>
<head>
      <title>css適用前</title>
</head>
<body>
      <h1>こんにちは！</h1>
      <p>はじめまして。私は Web開発を勉強中です。</p>
    
      <h2>趣味</h2>
      <p>カメラが好きで、犬の写真を撮っています</p>
      <img src="Luke(3).jpg" alt="犬の写真" width="300">

      <h2>作ったサイト</h2>
      <p><a href="https://t-higashi-findix.github.io/webpage-test/HTML&CSS-top/index.html" target="_blank">東のHTML & CSS, jQuery学習日誌</a></p>
</body>
</html>

      以下が適用後のものです。

      <!DOCTYPE html>
<html>
<head>
      <title>css適用後</title>
      <link rel="stylesheet" href="style.css">
</head>
<body>
      <h1>こんにちは！</h1>
      <p>はじめまして。私は Web開発を勉強中です。</p>
    
      <h2>趣味</h2>
      <p>カメラが好きで、犬の写真を撮っています</p>
      <img src="Luke(3).jpg" alt="犬の写真" width="300">

      <h2>作ったサイト</h2>
      <p><a href="https://t-higashi-findix.github.io/webpage-test/HTML&CSS-top/index.html" target="_blank">東のHTML & CSS, jQuery学習日誌</a></p>
</body>
</html>

      h1 {
      color: red;
}
p {
      font-family: "Yu Gothic";
      font-size: 1.2em;
}
h2 {
      font-family: "游明朝";
      color:blue;
      border-bottom: 2px solid blue;
}

      cssファイルが読み込まれてwebページが華やかになりましたね。

    • 内部スタイルシートの方法

      • HTMLファイルのheadタグ内でstyle要素を設けてCSSを埋め込む方法です。
      以下が使用例になります。結果は上のものと同じのため割愛します。

      <!DOCTYPE html>
<html>
<head>
      <title>css適用後</title>
      <style>
          h1 {
                color: red;
          }
          p {
                font-family: "Yu Gothic";
                font-size: 1.2em;
          }
          h2 {
                font-family: "游明朝";
                color:blue;
                border-bottom: 2px solid blue;
          }
      </style>
</head>
<body>
      <h1>こんにちは！</h1>
      <p>はじめまして。私は Web開発を勉強中です。</p>
    
      <h2>趣味</h2>
      <p>カメラが好きで、犬の写真を撮っています</p>
      <img src="Luke(3).jpg" alt="犬の写真" width="300">

      <h2>作ったサイト</h2>
      <p><a href="https://t-higashi-findix.github.io/webpage-test/HTML&CSS-top/index.html" target="_blank">東のHTML & CSS, jQuery学習日誌</a></p>
</body>
</html>

      コードの量が少ないときなど、シンプルなhtmlの場合はこの方法がよさそうですね。

    • HTMLの各要素にstyle属性を用いて直接記入する方法

      • HTMLファイルの各要素でstyle属性を用いてCSSを埋め込む方法です。
      以下が使用例になります。結果は上のものと同じのため割愛します。

      <!DOCTYPE html>
<html>
<head>
      <title>css適用後</title>
</head>
<body>
      <h1 style="color: red">こんにちは！</h1>
      <p style="font-family: Yu Gothic" style="font-size: 1.2em">はじめまして。私は Web開発を勉強中です。</p>
    
      <h2 style="font-family: 游明朝; color: blue; border-bottom: 2px solid blue">趣味</h2>
      <p style="font-family: Yu Gothic" style="font-size: 1.2em">カメラが好きで、犬の写真を撮っています</p>
      <img src="Luke(3).jpg" alt="犬の写真" width="300">

      <h2 style="font-family: 游明朝; color: blue; border-bottom: 2px solid blue;">作ったサイト</h2>
      <p style="font-family: Yu Gothic" style="font-size: 1.2em">
            <a href="https://t-higashi-findix.github.io/webpage-test/HTML&CSS-top/index.html" target="_blank">東のHTML & CSS, jQuery学習日誌</a>
      </p>
</body>
</html>

      1つ1つ指定するのは大変ですが、この方法では同じタグでも異なるCSSを指定したいときに使えそうですね。
      （例えばh1タグの見出しを2か所に使いたいが色だけは別々にしたい。など）


  5. 文字を装飾してみよう

    6. よく使うCSSのプロパティをいくつかご紹介します↓

    プロパティ 説明 例
    color 文字の色を指定 coler: blue;
    font-size 文字の大きさを指定 font-size: 24px;
    font-family フォントの種類を指定 font-family: sans-serif;
    font-weight 太字などの強調（boldなど） font-weight: bold;
    text-align 文字の配置（left,centerなど） text-align: center;

  6. 実例：簡単なデザイン付きページ

    7. では、CSSを使った簡単なデザイン付きページを作ってみましょう
    以下は私が作成したページになります。 

    <!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>スタイル付き自己紹介</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>こんにちは！</h1>
  <p class="intro">CSSでページにデザインを加えてみました。</p>
  <div class="main-wrapper">
    <img src="./Luke(3).jpg" id="dog">
    <p>我が家のワンコです。今年で6歳(?)になります。</p>
  </div>
</body>
</html>
    body {
  background-color: #f0f8ff;
  font-family: "Helvetica", sans-serif;
}

h1 {
  color: #2a7ae2;
  font-size: 36px;
  text-align: center;
}

.intro {
  color: #333;
  font-size: 18px;
  font-weight: bold;
  text-align: center;
}

.main-wrapper {
  text-align: center;
  border: double 5px #668ad8;
}

#dog {
  width: 400px;
}

  7. 練習課題

    8. 以下の項目を取り入れて、あなただけの自己紹介ページ（CSS適用版）を作ってみましょう

    • 背景色を変える
    • 見出しの色・サイズ・位置を変更
    • 段落にフォントや太さを指定
    • クラス名を使ってスタイルを個別に適用

  8. まとめ

    9. 今回のまとめです↓

    • CSSは「Webページのデザイン担当」
    • セレクタ { プロパティ: 値; }でスタイルを指定する
    • 色・サイズ・配置・フォントなどで文字の印象が変わる
    • 外部CSSファイルを使うと管理がしやすくなる
