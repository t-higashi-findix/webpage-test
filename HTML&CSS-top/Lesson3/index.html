Lesson 3：CSSの基本と文字装飾

ページに色やフォントを加えて、おしゃれにしてみましょう！

  1. はじめに

    2. HTMLでWebページの「構造」が書けるようになったら、次に学ぶべきはCSS（シーエスエス）。
    CSSは、文字の色や大きさ、背景などを設定できる「Webページのデザイン担当」です。
    このレッスンでは、CSSの基本的な使い方と、文字にスタイルを加える方法を学びます。
    ほんの少しCSSを加えるだけで、ページの印象が大きく変わりますよ


  2. CSSってなに？

    3. CSSは **Cascading Style Sheets（カスケーディング・スタイル・シート）**の略で、HTMLで作った構造に対して「見た目の指示」を与える言語です。
    例えば、文字の色やサイズ、フォント、行間、背景などを変えることができます。


  3. CSSの書き方（基本構文）

    4. CSSは○○.cssというファイル形式です。
    また、実際のCSSファイルの書き方は以下の構文を基本として作られています。


    
    セレクタ {
        プロパティ: 値;
    }

    
    h1 {
        color: red;
        font-size: 32px;
    }

    左が基本構文です。セレクタとはどのHTMLの要素にCSSを適用するか決める部分です。
    右側はCSSの書き方の一例です。 この例では読込先の.htmlファイルのh1セレクタに対して、
    colorをredに（文字の色を赤に）、font-sizeを32px（フォントサイズを32ピクセル）に設定するものとなっています。
    h1、つまり見出しを装飾しているという事です。


  4. CSSの適用方法

    5. CSSは3つの方法でHTMLに適用できます。

    • HTMLファイルとは別にCSSファイルをつくり、HTML文書内にそのファイルをCSSとして利用することを記す
      （外部スタイルシート）。
    • HTML文書のhead要素内にstyle要素を設け、HTML文書内にCSSを埋め込む
      （内部スタイルシート）。
    • HTMLの各要素にstyle属性を用いて直接記入する。

    引用：「HTML+CSSまとめページ/CSSの適用方法」(https://cc.musabi.ac.jp/kenkyu/cl1/html5/css_apply.html)


    この記事は外部スタイルシートを使ってCSSファイルを読み込むことでhtmlファイルを装飾しています。

    サンプルHTML↓

    
    <!DOCTYPE html>
    <html>
    <head>
        <title>自己紹介</title>
    </head>
    <body>
        <h1>こんにちは！</h1>
        <p>はじめまして。私は Web開発を勉強中です。</p>
        
        <h2>趣味</h2>
        <p>カメラが好きで、犬の写真を撮っています</p>
        <img src="dog.jpg" alt="犬の写真" width="300">

        <h2>作ったサイト</h2>
        <p><a href="https://t-higashi-findix.github.io/webpage-test/" target="_blank">東のHTML & CSS, jQuery学習日誌</a></p>

    </body>
    </html>

    便利知識：画像の幅と高さ

    画像サイズを指定したい場合は、widthやheight属性を使います。


  5. 文字を装飾してみよう

    6. よく使うCSSのプロパティをいくつかご紹介します↓

    プロパティ 説明 例
    color 文字の色を指定 coler: blue;
    font-size 文字の大きさを指定 font-size: 24px;
    font-family フォントの種類を指定 font-family: sans-serif;
    font-weight 太字などの強調（boldなど） font-weight: bold;
    text-align 文字の配置（left,centerなど） text-align: center;

  6. 実例：簡単なデザイン付き自己紹介ページ

    7. 今回のまとめです↓

    • <a href="URL">リンクテキスト</a>でリンクが作れる
    • <img src="画像のURL" alt="説明">で画像を表示できる
    • target="_blank"でリンク先を新しいタブに表示できる
    • widthやheightで画像サイズの調整も可能！

    次のページではhtmlで作成した文字に色を付けたり、大きくしたり、Webページの見た目をデザインするCSSの基礎を学びます！


  7. 練習課題

    8. 以下の項目を取り入れて、あなただけの自己紹介ページ（CSS適用版）を作ってみましょう

    • 背景色を変える
    • 見出しの色・サイズ・位置を変更
    • 段落にフォントや太さを指定
    • クラス名を使ってスタイルを個別に適用

  8. まとめ

    9. 今回のまとめです↓

    • CSSは「Webページのデザイン担当」
    • セレクタ { プロパティ: 値; }でスタイルを指定する
    • 色・サイズ・配置・フォントなどで文字の印象が変わる
    • 外部CSSファイルを使うと管理がしやすくなる

    次のページでは、「どこに何を置く？」を決めるために欠かせないボックスモデルとレイアウトの仕組みについて学んでいきます
