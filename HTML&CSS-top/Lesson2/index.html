Lesson 2：リンクと画像の使い方

リンクを貼ったり、画像を表示したり。

Webらしさが一気にアップします。

背景画像 著作者：rawpixel.com/出典：Freepik

    目次

  1. はじめに
  2. リンク（aタグ）の基本
  3. 画像（imgタグ）の基本
  4. 練習：自己紹介ページをグレードアップ
  5. 練習課題
  6. まとめ

  1. はじめに

    2. 前回は、HTMLの基本構造と「見出し」「段落」など、文字中心の内容を学びました。
    今回は、**Webページの魅力をグッと高める要素「リンク」と「画像」**について学びます。
    これができるようになると、他のページへ移動できたり、画像付きの自己紹介なども書けるようになります。


  2. リンク（aタグ）の基本

    3. リンクを貼るには<a>タグを使います。
    「anchor（アンカー）」の略で、リンクの出発点を意味します。
    具体的な書き方は以下の通りです。

    <a href="https://www.google.com">Googleを開く</a>

    作成したhtnlでリンクをクリックするとみなさんおなじみのGoogleが開いたと思います。
    各行の内容について簡単に解説を加えます。

    href
    		 ：href属性にリンク先のURLを記述する 
    <a>～</a>
    		 ：囲んだ部分がクリック可能なリンクになる

    コラム：リンクを新しいタブで開くには？

    リンク先を新しいタブで開くには<target="_blank">を追加します。

    <a href="https://www.google.com" target="_blank">Google（新しいタブで開く）</a>


  3. 画像（imgタグ）の基本

    4. 画像を表示するには、タグを使います。
    こちらは閉じタグが不要な「空要素」と呼ばれるタグです。

    <img src="sample.jpg" alt="サンプル画像">

    うまく読み込むと画像が表示されます（真ん中） うまく読み込めないとalt属性の内容が表示されます（右）


    タグ内の属性について簡単に解説を加えます。

    src属性
    		 ：画像ファイルの場所（sourceの略） 
    alt属性
    		 ：画像が表示されなかったときや、読み上げソフト用のテキスト（アクセシビリティ対応）

    実際のHTML例は以下の通りです。


    実際のHTML例

    <h1>私のペット紹介</h1>
    <p>うちのネコはとってもかわいいです。</p>
    <img src="cat.jpg" alt="うちのネコの写真">

    猫画像素材：フリー素材「ぱくたそ」


    次はLesson1の自己紹介をグレードアップしましょう！


  4. 練習：自己紹介ページをグレードアップ

    5. Lesson1で作った自己紹介ページに、画像とリンクを追加してみましょう！

    サンプルHTML↓

    <!DOCTYPE html>
    <html>
    <head>
    <title>自己紹介</title>
    </head>
    <body>
      <h1>こんにちは！</h1>
      <p>はじめまして。私は Web開発を勉強中です。</p>
        
      <h2>趣味</h2>
          <p>カメラが好きで、犬の写真を撮っています</p>
          <img src="dog.jpg" alt="犬の写真" width="300">

          <h2>作ったサイト</h2>
          <p><a href="https://t-higashi-findix.github.io/webpage-test/" target="_blank">東のHTML & CSS, jQuery学習日誌</a></p>

    </body>
    </html>

    便利知識：画像の幅と高さ

    画像サイズを指定したい場合は、widthやheight属性を使います。


  5. 練習課題

    6. 以下の項目を取り入れて、あなただけの自己紹介ページを作ってみましょう

    • 画像を1枚表示（例：旅行、ペット、風景など）
    • 好きなサイトへ飛べるリンクを1つ入れる
    • リンクは新しいタブで開くようにする

  6. まとめ

    7. 今回のまとめです↓

    • <a href="URL">リンクテキスト</a>でリンクが作れる
    • <img src="画像のURL" alt="説明">で画像を表示できる
    • target="_blank"でリンク先を新しいタブに表示できる
    • widthやheightで画像サイズの調整も可能！
