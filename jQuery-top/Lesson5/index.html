  4. スライドで開閉するボックス

    5. 実装するjQueryについておさらいしていきましょう。
    まずはスライドで開閉するボックスからです。

    この要素は、あらかじめ表示されている親要素と子要素を利用して作成します。
    手順としては、
    １．子要素を非表示にする
    ２．親要素に対してhoverやmouseenter、mouseleaveを利用してスライドアップ/ダウンを実装する
    といった手順でしたよね。

    それでは早速作ってみましょう！ Lesson4を参考にしてもいいと思います。
    （※補足までに、親要素のclassはそれぞれ「tab-top」「tab-html-css」「tab-jQuery」で、子要素はそれぞれ「none-top」「none-html-css」「none-jQuery」となっています）


  5. トップに戻るボタン

    6. 続いてトップに戻るボタンを完成させましょう。
    ...とはいっても現時点でトップに戻るの機能のみは満たせていますので、
    今回は戻る部分をゆっくり動かすように作成しましょう。

    この機能はclickメソッドとanimateメソッドを使って実装する機能でしたね。
    手順としては、
    １．return-Topをセレクタに指定しclickメソッドを作成する。
    ２．animateメソッドを１の中に作成し、ページの大元(bodyやhtmlタグ)をセレクタに指定する。
    ３．animateメソッドの第2引数に秒数（ミリ秒）を指定する。
    といったものでしたね。

    それではトップに戻るボタンを完成させましょう。


  6. スクロールしたらフェードイン

    7. 最後にスクロールしたらフェードインの機能です。
    今回はTopに戻るボタンをフェードインさせますが、せっかくなのでフェードインのついでにヘッダーの方をフェードアウトさせちゃいましょう。
    スクロールの際はscroll()メソッドで画面のスクロール位置を取得できましたよね。

    手順としては、
    １．ボタンはdisplay: none;を用いて、はじめは見えないようにしておく
    ２．windowをセレクタにしてscroll()メソッドを呼び出す。
    ３．scrollTop()メソッドを使ってスクロール量を取得する。
    ４．３を使ってスクロール量が一定を超えたらフェードアウト、超えていない場合はフェードインするような条件分岐を加える。
    （ヘッダーは逆です）

    それではスクロールしたらフェードインを完成させましょう。


  7. 回答例

    8. 以下に回答例を示します。 

    $(document).ready(function() {
  //スライドで開閉するボックス
  $('.tab-top').hover(
    function() {    // マウスを乗せるとスライドダウン
      $(this).find('.none-top').stop(true, true).slideDown(200);
    },
    function() {    // マウスを離すと元に戻る
      $(this).find('.none-top').stop(true, true).slideUp(200);
    }
  );
  $('.tab-html-css').hover(
    function() {    // マウスを乗せるとスライドダウン
      $(this).find('.none-html-css').stop(true, true).slideDown(200);
    },
    function() {    // マウスを離すと元に戻る
      $(this).find('.none-html-css').stop(true, true).slideUp(200);
    }
  );
  $('.tab-jQuery').hover(
    function() {    // マウスを乗せるとスライドダウン
      $(this).find('.none-jQuery').stop(true, true).slideDown(200);
    },
    function() {    // マウスを離すと元に戻る
      $(this).find('.none-jQuery').stop(true, true).slideUp(200);
    }
  );
  // トップに戻るボタン
  $('#returnTop').click(function () {
    $('body,html').animate({
      scrollTop: 0
    },500);  // 500ミリ秒で先頭までスクロールする
  });
  // スクロールしたらフェードイン（ヘッダーはフェードアウト）
  $(window).scroll(function() {
    if ($(this).scrollTop() > 50) {
      $('#returnTop').fadeIn();   // ボタンをフェードイン
      $('header').fadeOut();      // ヘッダーをフェードアウト
    } else {
      $('#returnTop').fadeOut();  // ボタンをフェードアウト
      $('header').fadeIn();       // ヘッダーをフェードイン
    }
  });
});
