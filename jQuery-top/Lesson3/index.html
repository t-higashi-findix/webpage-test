Lesson 3：要素の操作（表示/非表示・クラス切り替え）

要素の表示・非表示、クラスの切り替えを自由にしてみましょう。

背景画像 著作者：rawpixel.com/出典：Freepik

    目次

  1. jqueryでできる「要素の操作」とは？
  2. 表示・非表示の基本
  3. fadeで滑らかに表示・非表示
  4. クラスの追加・削除・切り替え
  5. 応用：2つの要素を切り替える
  6. まとめ

  1. jqueryでできる「要素の操作」とは？

    2. jQueryを使うと、HTMLの要素に対して次のような操作が簡単にできます：

    1. 表示/非表示
    2. クラスの追加・削除・切り替え
    3. 内容の書き換え（text / html）
    今回は特に 表示・非表示 と クラス操作 にフォーカスして学んでいきましょう！


  2. 表示・非表示の基本)

    3. jQuarynにおける表示・非表示では以下のメソッドを使うことで簡単に実装ができます。
    以下に表示・非表示に関するメソッドについて簡単に説明します↓

    表示・非表示に関するメソッド

    メソッド 説明
    .fadeIn() 要素を表示する
    .fadeOut() 要素を非表示にする
    .fadeToggle() 表示/非表示を切り替える
    
<p id="fade-text">ふわっと表示・非表示</p>
<button id="fade-btn">fade切り替え</button>

<script>
$(document).ready(function() {
    $('#fade-btn').click(function() {
    $('#fade-text').fadeToggle();
});
});
</script>


    <p id="msg">こんにちは！</p>
<button id="btn">クリックして変更</button>

<script>
 $(document).ready(function() {
  $('#btn').click(function() {
   $('#msg').text('こんにちは、jQuery！');
  });
 });
</script>

  3. クラスの追加・削除・切り替え

    4. jQueryでは見た目にかかわるイベントのみならず、クラスの追加や削除も行うことができます。
    以下のにクラスの追加や削除にかかわるメソッドについて簡単に説明します。

    クラスの追加・削除・切り替えにかかわるメソッド

    メソッド 説明
    .addClass クラスを追加する
    .removeClass() クラスを削除する
    .toggleClass() クラスの有無を切り替える

  4. 応用：2つの要素を切り替える

    5. toggleClassを使うことで2つの要素を切り替えているように表示することもできます。 

    <p id="msg1">こんにちは！</p>
<p id="msg2" style="display:none;">さようなら！</p>
<button id="switch-btn">切り替え</button>

<script>
 $(document).ready(function() {
  $('#switch-btn').click(function() {
   $('#msg1').toggle();
   $('#msg2').toggle();
  });
 });
</script>
  5. まとめ

    6. まとめです↓

    • .show() / .hide() / .toggleで表示状態を制御できる
    • .fadeToggle()などを使うとアニメーションが付くようになる
    • .addClass() / .removeClass() / .toggleClassでクラスの切り替えが可能

    次回はWebページをさらに華やかにできるアニメーションについて学びましょう！
