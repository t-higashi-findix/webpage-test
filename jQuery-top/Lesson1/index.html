  1. jQueryとは？

    2. jQueryは、JavaScriptをより簡単に・短く書くための軽量なJavaScriptライブラリです。
    複雑なイベント処理やアニメーションなどを、わずかなコードで実現できます。

    例えば... 

    // JavaScript         （JavaScriptは「//」でコメントアウトできる）
document.getElementById('box').style.display = 'none';

// jQuery
$('#box').hide();

    これはどちらも同じ、「"box"要素を隠す」といった内容になりますが、 jQueryが明らかにJavaScriptよりも少ないことが分かります。
    jQueryはこの分かりやすさとコードの簡潔さが魅力です。


  2. jQueryを使う準備

    3. jQueryを使うためにはjQueryのライブラリをHTMLファイルに読み込む必要があります。（左のコード）
    HTMLファイルにjQuery(javascriptファイル)を読み込むには以下のコードのように<script>タグを用いて、<head>要素内、または<body>の末尾に記述します。

    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>jQueryの導入</title>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
</head>
<body>
    <h1>Hello jQuery!</h1>

    <script>
    $(document).ready(function() {
        // jQueryの処理を書く場所
        console.log("jQueryが読み込まれました！");
    });
    </script>
</body>
</html>

    右のコードのようにhtmlファイル内にすべてのjQueryのコードを組み込む方法もありますね。
    アニメーションをあまり使わないWebページであれば十分に書き込める内容と記述量ですね。
    記述量が多い場合などはjsファイルをscriptタグで読み込む方法もあります。


  3. jQueryの基本文法

    4. jQueryを使う準備ができたので早速jQueryを使ってみましょう！
    ここではまずjQueryを使うには欠かせない$()と、よく使うセレクタの例について示します。

      $()の使い方

      $(セレクタ).メソッド();
      • $：jQueryオブジェクトの基本記号
      • セレクタ：操作したいHTML要素（CSSと同じ記述方法）
      • メソッド：操作内容（例：hide(),hover(),click()など）

  4. 簡単な操作例：文字の変更

    5. 簡単な操作例として文字の変更をしてみましょう。
    以下のコードを作成して実行してみましょう。 

    <!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>jQuery Lesson 1</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>はじめてのjQuery</h1>
    <button id="showBtn">クリックしてメッセージを表示</button>
    <p id="message">こんにちは、jQueryの世界へようこそ！</p>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <script src="script.js"></script>
</body>
</html>
    #message {
    display: none;
    margin-top: 20px;
    font-size: 20px;
    color: #2a7ae2;
}
    $(document).ready(function() {
    $('#showBtn').click(function() {
        $('#message').fadeIn();
    });
});

    記載したjQueryコード(jsコード)について説明します↓

    jQueryコードとその意味

    コード 意味
    $(document).ready() ページの読み込みが完了したら実行する
    $('#showBtn') idが showBtn の要素を取得
    .click(function(){ ... }) クリックされたときの処理を書く
    $('#message').fadeIn() メッセージをフェードインして表示

  5. よく使うメソッド（入門編）

    6. 以下にjQueryでよく使われる記述例について示します。

    jQueryコードの目的と記述例

    目的 記述例
    要素を取得 $('#id'), $('.class')
    クリック時の処理 .click(function(){ ... })
    表示/非表示 .show(), .hide(), .fadeIn(), .fadeOut()
    テキストを変える .text("新しい文字")

    要素を取得については$('#～')がidの取得で、$('.～')がクラスの取得になります。使い分けましょう。
    また、.click(function(){})については.on('click',function(){})などの書き方もあります。
    色々調べてみてはいかがでしょうか。


  6. jQueryのメリット

    7. jQueryには以下のようなメリットがあります。

    • 短いコードで直感的に書ける
    • 初学者にとってはJavaScriptよりも理解しやすい
    「特にJavaScriptよりも理解しやすい」については筆者がJavaScriptに触れたことがないことも起因しますが、
    「セレクタを指定して処理を書くだけ」という考え方がまとまっているので非常に使いやすいく感じました。


  7. 練習課題

    8. 練習問題として以下の機能も追加してみましょう

    • 非表示にするボタンを追加し、fadeOut()を使って消す


  8. まとめ

    9. まとめです↓

    • jQueryはJavaScriptを"直感的に簡単に"書けるライブラリ
    • $(document).ready()の中に処理を書くのが基本
    • $('#id')や$('.class')でHTML要素を操作できる
    • .click()を使ってクリックに反応させる
