    2. jQuaryでは、ほんの数行のコードで要素にアニメーションを加えることができます

  2. 作るもの：3種類のアニメーション

    3. 今回のレッスンでは次の３つのアニメーションを実装します！

    1. スライドで開閉するボックス

  3. スライドで開閉するボックス

    4. スライドで開閉するボックスとは本ページの上部にあるヘッダーの以下のようなアニメーションを指しています。↓

    ↑階層構造のWebページなどにこの方法は有効と思われます！

    実装方法

    1. 親要素の作成

      2. まずは要素がまとまったボックスを作成しましょう！
      具体的には上の動画の「トップメニュー」や「HTML&CSS」にあたる部分を作ります。
      以下のようなコードで早速要素を作成しましょう！ 

      <body>
  <div class="box-wrapper">
    親要素
  </div>
</body>
      .box-wrapper {
  background-color: blueviolet;
  color:chartreuse;
  text-align: center; /*テキストを中央揃え*/
  width: 30%;
  margin: 0 auto; /*要素を中央に配置*/
}

      ↑htmlファイルは<html>タグや<head>タグなどを省略して<body>タグの中身のみ記載しています。可視性の向上が目的です。
      色付けはcssファイルをお好みで弄ってください。

    2. 子要素の作成

      3. 次にボックスの中身（スライド開閉の部分）を作成しましょう。
      具体的な作り方として、box-wrapperクラスを持つ四角形を親要素として、その下に子要素を追加するイメージです。

      では、以下のようにコードを追加しましょう↓

      <body>
  <div class="box-wrapper">
    親要素
    <ul class="box-child">
      <li>子要素1</li>
      <li>子要素2</li>
      <li>子要素3</li>
    </ul>
  </div>
</body>
      .box-wrapper {
  background-color: blueviolet;
  color:chartreuse;
  text-align: center; /*テキストを中央揃え*/
  width: 30%;
  margin: 0 auto; /*要素を中央に配置*/
}
.box-child { /*li要素の点「・」を消す*/
  list-style: none;
  padding-left: 0;
  margin-left: 0;
}

      子要素1,2,3が下に並びましたね。ただ、このままでは表示されっぱなしなので、子要素を非表示にしましょう。

    3. 子要素の非表示

      4. 次に表示されている子要素を隠します。 隠し方は非常に簡単です。
      display: none;を子要素に追加するだけです。 早速追加してみましょう↓

      <body>
  <div class="box-wrapper">
    親要素
    <ul class="box-child">
      <li>子要素1</li>
      <li>子要素2</li>
      <li>子要素3</li>
    </ul>
  </div>
</body>
      .box-wrapper {
  background-color: blueviolet;
  color:chartreuse;
  text-align: center; /*テキストを中央揃え*/
  width: 30%;
  margin: 0 auto; /*要素を中央に配置*/
}
.box-child { /*li要素の点「・」を消す*/
  list-style: none;
  padding-left: 0;
  margin-left: 0;
  display: none;        //子要素を非表示にする
}

      ※htmlはそのまま

      子要素が隠れましたね。
      さて、ここからが本番です。jQueryを記述していきましょう。

    4. アニメーションの実装（スライドで開閉するボックス）

      5. 最後にマウスが乗ったり/離れたりしたらボックスを表示/非表示できるようにしましょう。
      マウスが乗った時/離れた時に動作するイベントはhoverや、mouseenterとmouseleaveでしたよね➡Lesson2
      今回はhoverを用いて実装してみましょう！
      今回のようにhoverを使う時は親要素のbox-wrapperを指定します。

      では、jsファイルを作成して次のように記述しましょう。↓

      $(document).ready(function() {
  $('.box-wrapper').hover(
    function() {
      $(this).find('.box-child').slideDown(200);
    },
    function() {
      $(this).find('.box-child').slideUp(200);
    }
  );
});

      $(this)は選択した要素の子要素を取得するために記述しています。
      また、.find('.box-child')で子要素の取得を実際に行って、.slide○○でスライドアップとスライドダウンを行っています。
      これでスライドで開閉するボックスが完成です！

  4. トップに戻るボタン

    5. トップに戻るボタンとは本ページの右下に出てくる、押したらページの最上部に移動するようなアニメーションを指しています。↓

    ↑長いWebページであればこの機能が役に立つ...かも？

    実装方法

    1. ボタンの作成&リンクの紐づけ

      2. まずはトップに戻るボタンを作成しましょう！
      具体的には上の動画の「Return to Top」にあたる部分を作ります。
      しかし、単にボタンを作成しただけでは機能を持ちません。 なのでaタグを使ってページトップへのリンクを作成します。
      また、トップに戻れるだけのページの長さも必要です。
      これらを踏まえて、以下のようなコードで要素を作成しましょう！ 

      <body>
  <p>
    text<br>
    (↑を好きなだけ繰り返す)
  </p>
  <p id="scroll-Top><a href="#">Topへ戻る</a></p>
</body>
      #scroll-Top {
    background-color: yellow;
    border: 1px solid blue;
    border-radius: 20px;
    padding: 10px;
    position: fixed;
    bottom: 25px;
    right: 25px;
}

      ↑index.htmlのtext<br> は、好きなだけ繰り返してページ横のスクロールバーが現れて少し動かせるくらいまで増やしてください。

      a href="#"によって"#"がリンク先となっていますが、"#"のみの記述ではページのトップを指す扱いとなります。
      この書き方を応用すると、例えば、
      a href="#example1"と設定するとページ内要素のid="example1"と指定した箇所にページの移動ができます。
      このような書き方をページ内リンクと呼びます。

      トップに戻るという機能をボタンに付与できたので、もう既にトップに戻るボタンは完成しています。
      しかし、このままではボタンを押したら一瞬でページが移動するので目が疲れますよね。

      という事でjQueryを使って、ページの移動を目が追える速さにしましょう。

    2. animateメソッドを使ったアニメーションの制御

      3. ページの移動をゆっくりにしましょう。
      以下のようにjsファイルを作成しましょう↓

      $(document).ready(function() {
  $('#scroll-Top').click(function () {
    $('body,html').animate({
      return-Top: 0
    },500);
  });
});

      ※htmlとcssはそのまま

      見覚えのないメソッドが現れましたね。
      今回使用したメソッドはanimateメソッドになります animateメソッドは使いこなせればページが一層華やかになります
      使い方を学びましょう

      animateメソッドとは

      animateメソッドとはその名の通りアニメーションを実装するためのメソッドです。
      animateメソッドでは、オブジェクトの移動や透過率などを変更できます。
      アニメーションを実装するためのメソッドは何個か存在しますが、簡単なアニメーションの場合、animateメソッドが一番使用されています。

      animateメソッドの書き方

      基本のコード

      基本のコードとしては、以下のように動かす要素名に対してparamsプロパティで指定したアニメーションを適用します。
      [duration], [easing], [complete]は省略可能です。

      $('動かす要素名').animate(params, [duration], [easing], [complete])

      使用例

      $(function() {
  $('動かす要素名').animate({
    '動かすアニメーション名': '動かす距離'
  });
});

      動かす要素名はclassやidで指定します。
      また、動かすアニメーション名はopacity(透明度)や、marginLeft（右に動かす）などの指定が可能で、動かす距離は100pxのように指定可能です。

      animateメソッドのプロパティ例

      animateメソッドでアニメーションに利用できるプロパティは、以下の4つです。

      1. params
      2. duration
      3. easing
      4. complete
      これらの役割と利用例を解説します。

      1. params
        2. 必須プロパティ。最終的に変化させるCSSのキーと値のマップ値を指定。基本的に数値で表せるもの。
        以下のようなものがある。
        width（幅）, height（高さ）, top（上）, left（左）, right（右）, bottom（下）, margin, paddng, fontSize（フォントサイズ）, opacity（透明度）

        実際の使用例は以下のようになる。
        $('#box').animate({
  'width': "100px",
});
      2. duration
        3. 省略可能なオプション。一般的には数値を指定する。1000を指定すると、1秒のアニメーションを行う。
        何も指定しないとnormalという指定になる。normal以外にもslowやfastという指定値も可能。
      3. easing
        4. 省略可能なオプション。値の変化を調節する独自関数を指定できる。swingとlinearという指定値が用意されている。
        swingが初期値。（swingは、始めと終わりは緩やかで途中は速めのアニメーションのこと）
        linearは 、一定スピードのアニメーションのこと。
      4. complete
        5. 省略可能なオプション。アニメーション終了時に実行する処理を指定できる。


      引用：TechAcademyマガジン


  5. スクロールしたらフェードイン

    6. スクロールしたらフェードインとは、本ページの右下の「Return to Top」がページをスクロールしたらフェードインするようなアニメーションを指しています。↓


    会社のHPなども「スクロールしたら○○」といったアニメーションを使っていますね。↓
    この場合は「スクロールしたらフェードイン（左の要素から順に）」でしょうか

    これらのようなアニメーションの作り方は、私の場合はスクロールの量によって制御しています。
    参考資料➡ 【jQuery入門】scrollTop()で画面のスクロール位置を取得・設定する方法！（侍エンジニア）

    スクロールに関したメソッドにscrollTop()があります。
    scrollTop()は画面をスクロールしたときの位置（スクロール量）を取得できるメソッドとなります。
    これにより、スクロールに対応して画面上部に戻るボタンを作成することができます。
    これを応用できれば以下のようなアニメーションも考えられるようになり、おしゃれなページが出来上がりますね。

    ではscrollTop()を使ったフェードインを4で作ったトップに戻るボタンを使って作ってみましょう

    実装方法

    1. 初期状態の作成

      2. ボタンははじめは見えない状態でなければなりません。
      そのため次のようなコードで、ボタンを見えなくしましょう
      実装はとても簡単でdisplay: none;をcssに書き足すだけです。

      #scroll-Top {
    background-color: yellow;
    border: 1px solid blue;
    border-radius: 20px;
    padding: 10px;
    position: fixed;
    bottom: 25px;
    right: 25px;
    display: none;
}

      右下の常に見えていたボタンが見えなくなりましたね。display: none;は指定した要素を完全になくしていると同じ状態です。
      簡略なサイト設計でなければさほど困りませんが、display: none;では次の要素でその場所が埋められるという事を頭の片隅に入れておいてください

    2. jQueryの作成

      3. お待たせしました。それではjQueryを作ってボタンをフェードインさせてみましょう！
      以下のコードをjsファイルに追加しましょう

      $(document).ready(function() {
  $('#scroll-Top').click(function () {
    $('body,html').animate({
      return-Top: 0
    },500);
  });

  $(window).scroll(function() {
    if ($(this).scrollTop() > 50) {
      $('#scroll-Top').fadeIn();  // ボタンをフェードインさせる
    } else {
      $('#scroll-Top').fadeOut();  // ボタンをフェードアウトさせる
    }
  });
});

      一般的にスクロール位置を取得するには、画面が上下に移動するスクロール量を連続的に知る必要があります。
      このようなケースで便利なのが「scroll()」イベントです。これらを組み合わせて、画面のスクロール位置を取得します！

      画面のスクロール位置を知りたいので対象の要素は「window」となります。 画面の全体を見ているといったイメージですね。
      そして、「scroll()」イベント処理内でscrollTop()を実行することで、画面が上下に移動する度に画面の最上部からのスクロール位置を取得できるということです
      スクロールによるイベントを複数個発生させたいときは「scroll()」イベント内で複数個の処理を書くようにしましょう。

      次に、先ほどから度々出てきている「$(this)」について。
      簡潔に述べると、thisとは「データ取得専用の変数」だそうです。
      この書きかたは複数の要素が同じ扱いとならないように"処理の切り分け"ができるそうです。
      詳しいことは下の資料を見てみてください。 少なくとも筆者はそのような認識です。
      参考資料➡【jQuery】$(this)の意味＆使い方について

      そして、$(this)の部分は全体で見るとif($(this).scrollTop() > 50)というif文になっていますね。
      つまり、先ほどのscrollTop()の説明と併せると、「スクロール位置が50を超えたらフェードインさせる」「そうでなければ（50以下なら）フェードアウトさせる（フェードインした後の処理）」
      といった処理が実装できました！

      これでアニメーションの完成です！おつかれさまでした！

  6. そもそもなぜjQuery？
