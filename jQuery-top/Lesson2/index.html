Lesson 2：イベント操作を使ってみよう

クリックやホバーで、Webページにちょっとした魔法をかけよう！

背景画像 著作者：rawpixel.com/出典：Freepik

  1. イベントとは？

    2. イベントとは、「クリック」や「マウスオーバー」、「キーボード入力」など、ユーザーのアクションに応じて発生する動作のことです。
    jQueryを使えば、このようなイベントに簡単に反応するコードを書けます。


  2. よく使われるイベント一覧

    3. jQueryでよく使われるイベントについて以下の表にまとめます。↓

    jQueryイベントの基本構文

    イベント名 説明
    click 要素がクリックされた時
    dbclick 要素がダブルクリックされた時
    mouseenter マウスが要素に乗った時
    mouseleave マウスが要素から離れた時
    hover mouseenterとmouseleaveを組み合わせたもの（まとめたもの？）
    change 値が変更された時（フォームなど）
    keydown キーが押された時
    submit フォームが送信された時

  3. 基本構文

    4. jQueryのイベントの基本構文は次のように書きます。 

    $(セレクタ).イベント名(function() {
    // 実行したい処理
});

    また、htmlに組み込む場合は次のように書きます。 

    <!DOCTYPE html>
<html lang="ja">
<head>
    <link rel="stylesheet" href="style.css">
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
</head>
<body>
    <h1>はじめてのjQuery</h1>
    <button id="showBtn">クリックしてメッセージを表示</button>
    <p id="message">こんにちは、jQueryの世界へようこそ！</p>
    <script>
    $(document).ready(function() {
        $('#showBtn').click(function() {
            $('#message').fadeIn();
        });
    });
    </script>
</body>
</html>
    #message {
    display: none;
    margin-top: 20px;
    font-size: 20px;
    color: #2a7ae2;
}

    また、セレクタには複数のイベントの割り当てもできます。 

    <!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <p id="hover-msg">ここにマウスを乗せてみて</p>
    <script>
    $(document).ready(function() {
        $('#hover-msg').mouseenter(function() {
            $(this).text('マウスが乗りました！');
        }).mouseleave(function() {
            $(this).text('また来てね〜');
        });
    });
    </script>
</body>
</html>
    p {
    border: 2px solid black;
    width: 200px;
}
  4. 入力フォームのイベント例

    5. 以下に入力フォームのイベントの作成例を示します。↓ 

    <!DOCTYPE html>
<html lang="ja">
<head>
    <link rel="stylesheet" href="style.css">
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
</head>
<body>
    <input type="text" id="name" placeholder="名前を入力">
    <p id="output">名前がここに表示されます</p>

    <script>
        $(document).ready(function() {
            $('#name').change(function() {
                let value = $(this).val();
                $('#output').text('こんにちは、' + value + 'さん！');
            });
        });
    </script>
</body>
</html>

    このコードは以下のような流れで出力を行っています。
    １．.val()イベントでフォームの値を取得
    ２．.change()イベントで値の変更に反応して書き換え


  5. まとめ

    6. まとめです↓

    • jQueryでは.clickや.changeなどのイベントメソッドを使うことで、
      ユーザーの操作に反応するインタラクティブなページを作れる。
    • 複数イベントの組み合わせなども可能

    次回は要素の操作（表示/非表示・クラス切り替え）について学びましょう！
